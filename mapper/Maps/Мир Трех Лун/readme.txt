О шкурах (или по научному - о "темах")
-----------------------------------------------------------------------------
Структура файла, описывающего тему (шкуру):

[Skin0]                 - обязательная секция. она может быть единственной
Description=Def 32x32   - описание карты. для вывода в список в тулбаре
Picture=DefRooms.bmp    - файл с примитивами для рисования комнат
Icons=DefIcons.bmp      - файл с иконками, выводимыми на комнатах
NorthRect=10,0,22,8   -+- прямоугольники, задающие область примитива, в
SouthRect=10,25,22,32 -|  которой расположен проход (дверь, стена) на
WestRect=0,10,8,22    -|  север, юг, запад и т.д. 
EastRect=25,10,32,22  -|  Примитивы при загрузке разбиваются на эти куски
UpRect=9,9,16,16      -|  и потом выводятся в зависимости от того есть
DownRect=17,17,24,24  -+  проход (дверь) или нет в соответствующем направлении
IconPoint=8,8           - точка, в которую выводится левый верхний угол иконки
FillPoint=14,20         - точка пола, с которой начинается заливка цветом

[Skin1]                 - дополнительная секция. используется чтобы задать
Description=Small 16x16   альтернативный вид (цветовую гамму) или другие
Picture=Rooms16.bmp       размеры комнат (типа масштабирование)
Icons=
NorthRect=5,0,11,5        дополнительных секций может быть сколько угодно
SouthRect=5,12,11,16      Их имена должны быть Skin1, Skin2, Skin3 и т.д.
WestRect=0,5,5,11
EastRect=12,5,16,11
UpRect=5,5,8,8
DownRect=8,8,12,12
IconPoint=4,4
FillPoint=7,10

-----------------------------------------------------------------------------
Очень желательно, чтобы основная секция описывала тему с размером комнат
32х32 пикселя и с иконками размером 16х16.
Иконки основной секции используются для отображения в меню, поэтому другой
размер нежелателен.

Если указать несуществующий файл для комнат, или не указывать его совсем,
то будет использована внутренняя картинка. Есть смысл указать только файл для
иконок. Тогда будут подгружены иконки, а комнаты останутся по умолчанию.
В этом случае координаты прямоугольников и точки можно не задавать. Файл будет
иметь такой вид:

[Skin0]
Description=Мои иконки
Picture=
Icons=MyIcons.bmp

Если вы укажете несуществующий файл иконок для дополнительных секций, то
иконки выводиться не будут. Если указать левый файл или не указать вообще для
основной секции, то будут использованы иконки по умолчанию.

Желательно делать иконки соразмерными комнатам (обычно в 2 раза меньше), но это
не обязательно. Также необязательно лепить их ровно по центру комнаты.

Учтите, что иконки в разных файлах для одной темы могут быть разными по
размеру, но должны быть одинаковыми по содержанию, и их количество должно
совпадать. Иначе выбирая из меню одну иконку, вы получите в комнате совершенно
другую. 

Файл DefRooms.bmp содержит стандартную заготовку для темы.
Файл состоит из 26 картинок размером 32х32 пикселя соединенных в ленту.
Примитивы должны в точности совпадать по назначению с теми, что в этом файле.
Картинки используются для следующего:
1  - пол комнаты
2  - углы комнаты
3  - выходы отсутствуют (невидимы) и через них не ходили
4  - выходы есть, через них не ходили
5  - есть выходы с дверьми, через них не ходили
6  - выходы отсутствуют (невидимы) но через них ходили
7  - выходы есть, через них ходили
8  - есть выходы с дверьми, через них ходили
9  - есть триггера на проход в любом из направлений
10 - есть порталы в другие зоны или комнаты при выходе в любом из направлений
11 - коридор с севера на юг
12 - коридор с запада на восток
следующие 12 картинок - то же самое, но в контурном виде для отображения
верхнего уровня.
последние две картинки
25 - маркер текущего положения
26 - маркер помеченной комнаты (при потере трекинга, поиске и т.п.)

Если у вас на экране после загрузки шкуры получается белиберда, значит вы
ошиблись, рисуя файл с примитивами, или неверно задали координаты
прямоугольников, по которым изображение нарезается на отдельные куски.

В общем, надеюсь что все понятно. Если ничего не понятно, то лучше за рисование
шкур не браться.
